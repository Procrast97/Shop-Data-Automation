ℹ How to use this page
  1. Select the URL you want to edit from the left dropdown — the "New URL" field on the right auto-fills with the current value so you can see what you're changing.
  2. The Shop Location dropdown updates automatically to show locations for that URL. Select the one you want to rename — the "New Shop Location" field auto-fills.
  3. Edit only the field(s) you want to change, then click Save Changes. Leave a field as-is if you don't want to change it.

Note: this page only corrects names and addresses. To delete a record entirely, use the Remove page.
