ℹ How to use this page
  • Select Add Shops to register a new shop location under an existing or brand-new POS URL.
  • Select Add Items to manually record a single sale transaction against an existing shop location.
Add Shops — field guide:
  • URL Option: pick an existing URL from the dropdown, or switch to "Add New URL" and paste your POS link (must follow the format shown).
  • Shop Name: a short uppercase abbreviation for the shop, e.g. VD. This groups all locations under one brand.
  • Location: the physical store name, e.g. Central or Causeway Bay. Submit once per location.
Add Items — field guide:
  • Date of Sale: enter in DD/MM/YYYY format, e.g. 15/05/2026.
  • Shop Location: select the specific location this sale occurred at.
  • Price / Quantity: enter numeric values only (no currency symbol).
