Remove

Delete a shop location, an entire URL group, or a single recorded sale item. All deletions are permanent.

ℹ How to use this page
Remove URL / Shop (left form)
  • Select a URL from the dropdown, then tick Remove URL to delete the entire URL and every shop/location beneath it. This cannot be undone.
  • Or select a Shop Location and tick Remove Location to delete only that single location and its sales records.
  • You cannot tick both checkboxes at once.
Remove Item (right form)
  • Select a Shop Name — the location list will update automatically.
  • Pick the Shop Location, then choose a Date — the item list will populate with records for that date.
  • Select the specific Item and click Remove. A confirmation prompt will appear first.
