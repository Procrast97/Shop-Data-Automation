▶ How It Works

In your own account, the setup flow is:

  1. Click Add URL/Shop and register your POS URL and shop abbreviation, then add each physical location.
  2. Return to Add URL/Shop, select Add Items, and log your daily sales against the correct location.
  3. Head to Reports to filter by shop and date range, view a live summary, then export to Excel or PDF.
  4. Use Amend to correct any typos in URLs or location names at any time.
  5. Use Remove to delete a specific location, an entire URL/shop group, or a single recorded item.

Add URL / Shop

Register a new POS URL and give it a short abbreviation. Then add one or more physical store locations under it. You can also manually log individual sale transactions here using the Add Items toggle.

Remove URL / Shop

Delete a specific shop location, an entire URL with all its shops, or remove a single logged sale item. Deletions are permanent — use with care.

Amend

Correct a URL address or a shop location name after the fact. Select the existing record from the dropdowns, edit the text field that auto-fills, and save.

Reports

Filter sales by shop, location, and date range to instantly see totals for transactions, quantity, and revenue. Export the result to Excel or PDF for sharing.
