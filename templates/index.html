How It Works

  1. Register / Sign In — create your account or use the demo credentials above.
  2. Add a Shop — link a POS URL, give it an abbreviation (e.g. VD), and add one or more physical locations (e.g. Central, Causeway Bay).
  3. Log Sales Items — on the same Add page, switch to "Add Items" to manually record individual sale transactions against any shop location.
  4. Generate a Report — filter by shop, location, and date range to view a live summary table with totals, then export to Excel or PDF.
  5. Amend or Remove — correct a URL/location name, or delete outdated records, at any time.
